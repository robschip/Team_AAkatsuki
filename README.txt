Unserer Code(notebooks) in der Reihenfolge durchlaufen lassen:
1. step1_weather_data.ipynb
2. step2.1_preprocess_and_descriptive_analysis.ipynb
3. step2.2_greographical_demand.ipynb
4. step3_clustering.ipynb
5. step4_predictive_analytics.ipynb

Damit die notebooks alle ausführbar sind, müssen sie folgendes tun:
Die Daten unsere Gruppe: "chicago_2019.csv" zu dem Ordner hinzufügen
und die Wetterdaten "weather_hourly_chicago.csv" ebenfalls.
Dann muss einmal die Datei "step2.1_preprocess_and_descriptive_analysis.ipynb" ausgeführt werde, welche ein "processed_dataframe.csv" erstellt, welcher in anderen Notebooks verwendet wird.
Nachdem diese Datei erstellt wurde, können sie die anderen Notebooks problemfrei ausführen.
Das erste Notebook ("step1_weather_data.ipynb") ist nicht von dem "processed_dataframe.csv" abhänging und kann auch sofort ausgeführt werden.
